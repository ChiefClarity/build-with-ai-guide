# Build With AI — A Parent's Guide

A free, interactive guide for parents on building real projects with their kids using AI (Claude Code). The first guide walks through building a kid's online business end to end.

- `index.html` — the main guide
- `dashboard-demo.html` — an interactive demo of the business dashboard the kids use

Static site, no build step. Deployed on Vercel; every push to `main` auto-deploys.
